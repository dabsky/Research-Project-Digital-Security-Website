Spyware

What is Spyware?

According to google, “Spyware is a software that enables a user to obtain covert information about others computer activities by transmitting data covertly from their hard drive”. In other words, Spyware is a malicious malware that does not cause any damage or spread anything to other systems. It mainly focuses on the users themselves and it is controlled by another user. This user will try to steal private information from this user to sell. Spyware started to appear or be used in the early 2000’s.

Ways It Affects Computers

Spyware is used Mainly to sell information. It is not directly affect the computer, but the user.

Ways You Can Get It
  • Clicking onto a link
  • Entering sketchy internet websites
  • Downloading a program from a unknown website
  • Clicking on pop-up ads
  • Answering texts from unknown numbers
Symptoms

There are many ways to spot this malware. Such as noticing if your device runs slower than usual, or if it freezes up and crashes a lot. If there are pop-up ads on your homepage and not a website, then you most likely have spyware. Also if there is home browser changes, for example, a new icon, or error messages on normal links.
