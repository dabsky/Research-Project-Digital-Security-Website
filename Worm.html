Worm

What is a Worm?

A computer worm is a malware that reproduces itself and spreads over network connections. It does this by exploiting parts of an operating system that are automatic and invisible to the user. It’s different to a virus in that a virus requires human interaction, and a worm can spread automatically without a person doing anything.

Ways it Affects Computers
  • computer having performance issues
  • System freezes/crashes unexpectedly (these issues happen because worm often uses large amounts of bandwidth, so affected servers, networks, and individual systems are often overloaded and stop responding)
  • Unusual sounds/images
  • Appearances of unfamiliar files
  • Email messages sent to contacts that the user didn’t actually send (this also refers to email worms.) These work by creating and sending outbound messages to all the addresses in a user's contact list. The messages include a malicious executable file that infects the new system when the recipient opens it
Symptoms
  • Keep an eye on your hard drive space (because worms replicate again and again, they often take up large amounts of hard drive space.)
  • Monitor speed and performance
  • Be on the lookout for missing or new files (worms can delete files on your computer and then replace them with something else. If you see new files or notice some that are missing, it may be because of a worm).
Removal
The safest and most effective way to disinfect a computer that has been infested by a worm is by using a dedicated removal tool.
Ex: Avast! Virus cleaner - Download this, and this virus cleaner will check if a worm is running, and terminate the process if necessary. It will remove any tracks of a worm.
Prevention
  • Good cybersecurity hygiene
  • Install operating system updates and software patches
  • Use firewalls (will help reduce access to systems by malicious software)
  • Use antivirus softwares (prevents malicious software from running)
  • Use encryption ( protects sensitive data stored on computers, servers, and mobile devices
  • Update all antivirus signatures
  • Confirm that the operating system and all applications are up to date and patched

    • Conclusion

    A computer worm is sneaky in that it doesn’t need human intervention to attack a system, but instead it does that on its own by replicating itself into the system, and destroying what it can. For those reasons it can be difficult to remove a computer worm. At times the only thing possible left to do is to wipe a system entirely, but there are many other ways it can be prevented, and one can start by being aware of the ways it attacks a system.
